Content

The abstraction is made of the following files:

  • evk1101.h:
    • a preprocessor mapping of the on-board AT32UC3B pins connections to the on-board hardware resources.
  • led.c and led.h:
    • a specific set of functions to easily control the EVK1101 LEDs.
